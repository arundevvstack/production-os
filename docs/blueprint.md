# **App Name**: DP Media OS

## Core Features:

- Multi-Tenant Admin & User Management: Comprehensive system for company onboarding, user authentication via Firebase, role-based access control (RBAC), and dynamic module enablement. Ensures data isolation per tenant with super admin global oversight and a subscription-based feature gating.
- Media Production Project Management: Centralized tools for managing pre-production, production days, and post-production phases, including task assignments, budget tracking, and linking associated talents to projects.
- Talent Network & Marketplace: A marketplace for talent profiles (Actors, Models, Influencers) featuring detailed profiles with categories, social metrics, rates, availability calendar, portfolio management, and booking functionalities.
- Sales CRM & Proposal System: Manage leads through pipeline stages (Lead, Contacted, Meeting, etc.), track sales activities, and generate proposals using templates, including an AI tool for generating proposal content placeholders.
- AI Market Research Intelligence: Leverage AI to analyze market trends, competitor activity, and content trends for specified industries and locations, providing an AI tool to suggest optimal pitch angles and identify opportunity scores.
- Automated Invoicing & Google Integration: Generate multi-line item invoices with GST support, auto-numbering, status tracking, PDF generation, and automated upload to Google Drive, with line item synchronization to Google Sheets upon finalization.
- Dynamic Dashboard & Reporting: A modular dashboard providing an overview of key performance indicators such as active projects, revenue summaries, pending invoices, sales pipeline value, and talent booking counts, with alerts for overdue items.

## Style Guidelines:

- Primary Color: A deep, professional blue (#16389C) that conveys trust and corporate stability, ideal for core branding elements and active states.
- Background Color: A subtle, cool light grey-blue (#F0F1F4), derived from the primary hue but heavily desaturated to provide a clean, uncluttered canvas for complex data displays.
- Accent Color: A clear, vibrant cyan-blue (#42B2D7) provides energetic contrast and is used sparingly for call-to-actions, highlights, and important notifications.
- Headline Font: 'Space Grotesk' (sans-serif), chosen for its modern, slightly technical feel, suitable for titles, navigation, and distinct UI elements. Body Text Font: 'Inter' (sans-serif), for its excellent readability and neutral, objective appearance, ideal for large volumes of text, data tables, and form fields.
- Employ a consistent set of line-art or minimalist solid icons that are scalable vector graphics. Icons should be clear, intuitive, and integrate seamlessly with the modern, functional aesthetic of the OS.
- Implement a modular, responsive grid-based layout for dashboards and content areas, ensuring adaptability across devices. A dynamic, permission-based sidebar facilitates navigation within the multi-tenant architecture.
- Subtle and functional animations are used to enhance user experience, providing visual feedback for state changes, module transitions, and data loading, without being distracting or slowing performance.
