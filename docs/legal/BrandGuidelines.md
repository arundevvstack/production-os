# DP Creative OS — Brand Guidelines

## 1. Core Identity
DP Creative OS represents the intersection of enterprise reliability and boundless creative generation. It is the operating system for the next generation of AI-native film and media production.

## 2. Typography
- **Primary:** Inter (Used for all UI elements, dashboards, and tooltips).
- **Secondary / Scripting:** Courier Prime (Used strictly for Script Studio and screenwriting inputs to maintain industry standard formatting).

## 3. Color Palette
- **Background:** High-contrast Dark Mode (Tailwind `zinc-950`).
- **Primary Accent:** Vivid Indigo (`indigo-500` to `indigo-600`) to indicate AI generation actions or primary CTA buttons.
- **Success:** Emerald green (used for Asset Approvals and 100% completion metrics).
- **Error:** Rose red (used for destructive actions or Generation Failures).

## 4. Logo Usage
The logo should always be displayed with ample whitespace. Do not overlay the logo on busy AI-generated backgrounds without a solid dark-mode container.
